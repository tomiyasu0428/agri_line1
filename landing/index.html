“一瞬見て分かる” 直進ガイド

左右のズレだけに集中。Driveモードで矢印・色・ターゲット点を提示。スマホだけで今すぐ使える軽量Webアプリ。

デモを試す GitHub 使い方

特長

  • Driveモード: 矢印と背景色でズレ量を即判断（Wide/Normalしきい値）
  • キャロット(◎): 2秒先のターゲット点に自車を重ねるだけ
  • ロバスト更新: watch停止時は自動でpollingにフォールバック
  • PWA対応: オフラインでも軽快（nocache検証版あり）

使い方（3分で開始）

  1. 「計測開始」→ その場で A点 設定
  2. 10〜30m直進 → B点 設定
  3. 最寄ライン で合わせ、必要なら Driveモード をON

スマホGPSは ±数m〜十数m。狭い作業幅では“目安用途”に留まります。

外部GPSでさらに安定

iOS/AndroidともにOS統合GNSS（Dual XGPS160、Garmin GLO 2、Bad Elf など）で精度向上が期待できます。詳細は 外部GPS拡張メモ をご覧ください。
