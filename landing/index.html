“一瞬見て分かる” 直進ガイド

左右にどれだけズレたかを、矢印と色だけで一目で判断。スマホだけで今すぐ使える、直進を助けるアプリです。

デモを試す GitHub 使い方

特長

  • 矢印と色で一目で分かる: 左右のズレが大きいほど矢印が太くなり、色で「OK/注意/戻す」を表示
  • 目標の丸（◎）: 少し先に出る丸に、車体を重ねるだけでまっすぐ走れる
  • 自動で取り直し: 位置が止まっても自動で再取得して動作を継続
  • スマホでそのまま: アプリのようにホーム画面からすぐ起動。軽快に動作

使い方（3分で開始）

  1. 「計測開始」→ その場で A を記録（出発点）
  2. 10〜30mまっすぐ進んで B を記録（向きを決める）
  3. 最寄ライン（いちばん近い目標線）に合わせ、必要なら Driveモード をON

スマホの位置は ±数m〜十数mほどズレます。狭い幅では“目安”としてお使いください。

外付けGPSでさらに安定

市販の外付けGPS（例: Dual XGPS160、Garmin GLO 2、Bad Elf など）をつなぐと、位置のブレが小さくなります。詳しくは 外部GPSのメモ をご覧ください。
